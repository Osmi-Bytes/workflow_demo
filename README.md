# Simple Todo App

A lightweight Todo app built with vanilla JavaScript, HTML, and CSS.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter by all, active, and completed
- Clear all completed tasks
- Persist tasks in local storage

## Run

1. Open this folder in VS Code.
2. Start a local server (for example with the Live Server extension) and open `index.html`.
3. Use the UI to manage your tasks.

## Project Files

- `index.html` - app structure
- `styles.css` - styling and responsive layout
- `app.js` - app behavior and local storage logic
